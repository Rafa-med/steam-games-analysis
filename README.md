# Steam Games Analysis

This project explores a dataset of Steam games using data analysis and visualization techniques.

## Objectives
- Analyze game popularity (CCU)
- Explore the relationship between reviews and player engagement
- Understand pricing distribution of games
- Identify patterns in user behavior

## Tools
- Python
- Pandas
- Seaborn
- Plotly

## Dataset
Steam Games Dataset (SteamSpy API):  
https://www.kaggle.com/datasets/muhammadaqeelkabir/steam-games-dataset-steamspy-api?resource=download

## Key Insights
- Popular games tend to receive more positive reviews
- A small number of games concentrate most active players
- Most games are low-priced or free
- Higher engagement is associated with better user ratings
